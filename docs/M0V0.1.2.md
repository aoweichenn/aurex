# Aurex M0V0.1.2

M0V0.1.2 focuses on usability and visible self-hosting progress.

## Added

- `--check` CLI mode.
- `--emit=c` explicit C emission mode.
- `--help` CLI output.
- English usage guide: `docs/USAGE.en.md`.
- Chinese usage guide: `docs/USAGE.zh.md`.
- Self-hosting plan: `docs/SELFHOST.md`.
- Visible `selfhost/` directory.
- M0 selfhost seed: `selfhost/src/m0c_seed.ax`.
- Runtime placeholder: `selfhost/runtime/runtime.c`.
- Bootstrap chain verification script: `tools/bootstrap_chain.sh`.
- More semantic regression tests for opaque structs, returns, while conditions,
  and break/continue placement.

## Current Selfhost Status

The project is not fully self-hosted yet. M0V0.1.2 makes the path explicit and
tested:

```sh
tools/bootstrap_chain.sh
```

This verifies:

- the production Stage0 C++ compiler can compile the M0 selfhost seed;
- the standalone bootstrap compiler can compile the hello subset;
- both generated programs execute correctly.
