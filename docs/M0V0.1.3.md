# Aurex M0V0.1.3

M0V0.1.3 focuses on design documentation and day-to-day developer ergonomics.

## Added

- `--emit=tokens`, `--emit=ast`, and `--emit=check` aliases.
- Source-line diagnostics with caret markers.
- English architecture design document: `docs/DESIGN.en.md`.
- Chinese architecture design document: `docs/DESIGN.zh.md`.
- Semantics document: `docs/SEMANTICS.md`.
- C backend design document: `docs/C_BACKEND_DESIGN.md`.
- First selfhost lexer sequence smoke program: `selfhost/src/lexer_smoke.ax`.

## Kept Explicit

Self-hosting remains a top-level project goal, but M0V0.1.3 is still not a full
self-hosted compiler. The tested chain is:

```sh
tools/bootstrap_chain.sh
```

The next concrete self-hosting milestone is expanding
`selfhost/src/lexer_smoke.ax` into an M0 lexer implementation that can match the
C++ lexer token dump on a golden corpus.
